Stream your headset to your PC. Put your headset on before clicking Start Stream.
Start Stream
Oculus Casting
Disable Proximity Sensor
Always On Top
Make scrcpy window always on top (above other windows).
Redirect audio from the headset
Redirects audio from the headset to computer speakers.
Bit Rate
Encode the video at the given bit-rate, expressed in bits/s. Unit suffixes are supported: 'K' (x1000) and 'M' (x1000000).
Crop
Crop the device screen on the server. The values are expressed in the device natural orientation (typically, portrait for a phone, landscape for a tablet). Any MaxSize value is computed on the cropped size. The format is width:height:left:top.
Quest 2 Crop
Quest 1 Crop
Please note the Quest Pro uses two slightly angled screens and therefore there isn't a crop setting that works out of the box, since the video would need to be corrected after the fact.
Fullscreen
Start in fullscreen.
No Input
Disable device control (mirror the device in read-only).
Max FPS
Limit the frame rate of screen capture. 0 is unlimited.
Max Size
Limit both the width and height of the video to value. The other dimension is computed so that the device aspect-ratio is preserved. 0 is unlimited.
To use ScrCpy on mac you will need to install Homebrew and then install scrcpy with brew install scrcpy

To use ScrCpy on Linux you will need to install snap package manager with sudo snap install scrcpy
